# gsc-analyst-backend

Next.js backend for Cloud Run deployment.

## Required env

- `CLERK_SECRET_KEY`
- `FRONTEND_APP_URL`
- `GOOGLE_OAUTH_CLIENT_ID`
- `GOOGLE_OAUTH_CLIENT_SECRET`
- `GOOGLE_OAUTH_REDIRECT_URI`
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `ANTHROPIC_API_KEY`

## Run

```bash
npm install
npm run dev
```
